# Logo Evaluation Rubric

Score each criterion 1-5. Total possible: 25 points.

## 1. Text Legibility

How readable is the brand name or lettering in the logo?

| Score | Description |
|-------|-------------|
| 5 | Every character perfectly formed, instantly readable at any size |
| 4 | All characters readable, minor imperfections that don't affect comprehension |
| 3 | Readable with effort, some characters slightly malformed or unclear |
| 2 | Partially readable, multiple characters distorted or merged |
| 1 | Text unreadable, gibberish characters, or missing entirely when expected |

If the logo intentionally has no text (pure symbol), score based on how well the symbol communicates without text. A strong standalone symbol scores 4-5.

## 2. Simplicity and Memorability

Can someone sketch the logo from memory after seeing it once?

| Score | Description |
|-------|-------------|
| 5 | Iconic, minimal elements, immediately memorable (think Apple, Nike) |
| 4 | Clean design with few elements, easy to recall |
| 3 | Moderate complexity, recognizable but hard to reproduce from memory |
| 2 | Too many elements competing for attention, cluttered |
| 1 | Chaotic, no clear focal point, impossible to remember |

## 3. Color Harmony

Do the colors work together and support the brand feeling?

| Score | Description |
|-------|-------------|
| 5 | Colors are cohesive, evoke the right mood, would work in both color and monochrome |
| 4 | Good palette, minor adjustments would perfect it |
| 3 | Acceptable colors but feeling is off, or too many competing colors |
| 2 | Colors clash or feel wrong for the brand/industry |
| 1 | Garish, random, or completely inappropriate color choices |

## 4. Scalability

Will the logo work from favicon (16px) to billboard?

| Score | Description |
|-------|-------------|
| 5 | Clean shapes, strong silhouette, no fine details that would disappear at small sizes |
| 4 | Works at most sizes, minor details might be lost at very small scales |
| 3 | Reasonable at medium sizes, but fine details or thin lines would cause problems at extremes |
| 2 | Relies on detail that breaks below medium size |
| 1 | Only works at one specific size, intricate detail or thin lines throughout |

## 5. Brief Alignment

Does the logo match what was asked for?

| Score | Description |
|-------|-------------|
| 5 | Nails the brief: correct style, mood, industry feel, requested elements all present |
| 4 | Closely matches brief with minor deviations |
| 3 | Partially matches, captures some elements but misses others |
| 2 | Loosely related to brief, significant mismatches |
| 1 | Completely off-brief, wrong style/mood/elements |

## Scoring Summary

Present results as a table:

```
| # | File | Text | Simple | Color | Scale | Brief | Total | Notes |
|---|------|------|--------|-------|-------|-------|-------|-------|
| 1 | ...  | 4    | 5      | 4     | 5     | 4     | 22    | ...   |
```

**Thresholds:**
- 20-25: Strong candidate for refinement
- 15-19: Promising direction, needs work
- 10-14: Interesting elements but significant issues
- Below 10: Discard, try different approach
